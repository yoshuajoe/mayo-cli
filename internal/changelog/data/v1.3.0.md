# Mayo v1.3.0 - AI Data Enhancer & UX Polishing

### 🧪 NEW: AI Data Enhancer
- **Background Processing**: You can now spin up background workers to enrich your SQLite data.
- **Incremental Batching**: Processes data in batches to keep tokens low and costs predictable.
- **Idle Mode**: Workers can wait for new data to arrive in your tables and process it automatically.
- **Interactive Wizard**: Run `/enhance start` without flags for a guided setup.

### ⚡ UX Improvements
- **Strict Dataframe Validation**: Commands like `/enhance` now correctly require an active dataframe to prevent context errors.
- **Interactive Selection**: Improved selection UI for databases and tables.
- **Better Error Messages**: Clearer instructions when a command is used outside its intended context.
- **Auto-Suggestions**: Fixed command typos with better fuzzy matching.

### 🐞 Bug Fixes
- Fixed a loop issue where running certain slash commands could restart the session.
- Fixed `survey.Input` type mismatches for numerical settings.
